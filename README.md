# Inventory Management System
Final Project for Python Foundation Course (ReDi School Fall Semester 2025). Create an expandable inventory tracking program that: 
1) View Inventory,
   a) current inventory status is displayed,
   b) critical inventory level for each items is displayed
   c) analysis on restock: Yes or No
2) Creating Sales Order
3) View Sales Orders
4) Crea Purchase Order
5) Analysis
   a) Sales activity
   b) Cost Effectiveness (??)
   c) Vendor Pricing (??)
6) Exit
